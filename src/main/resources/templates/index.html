What is GitOps?

GitOps is a development methodology that uses Git as a source of truth for infrastructure and application deployment. With GitOps, developers can version control their infrastructure and application configuration, and use pull requests to manage changes to their environments.

How does Config Sync work?

Config Sync is a tool provided by Google Cloud that enables GitOps for Kubernetes. It automatically syncs Kubernetes objects (such as Deployments, Services, and ConfigMaps) defined in a Git repository to a Kubernetes cluster, ensuring that the cluster matches the desired state defined in the repository.
